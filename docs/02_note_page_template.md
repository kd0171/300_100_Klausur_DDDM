## 過去問関連の書き方

悪い例:

```md
過去問に出た。
```

良い例:

```md
- SS24 Aufgabe 5a と関連。
  - 関連点: パッケージ配送問題をMDPとして記述する設問。
  - 必要理解: 決定時点、状態、決定、費用、後決定状態、外生情報、遷移、目的関数を具体例に対応付けること。
```

## 試験対策ポイントの書き方

過去問だけに依存せず、次も含める。

- 定義を言えること。
- 図を説明できること。
- 具体例に適用できること。
- 式の記号の意味を言えること。
- 手法の強み・弱みを比較できること。
- 「なぜそのモデルを選ぶか」を説明できること。
