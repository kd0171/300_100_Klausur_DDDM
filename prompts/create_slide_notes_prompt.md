# Prompt: Markdown学習ノート作成

次のDDDM講義スライドについて、Markdown学習ノートを作成してください。

## 方針

- 1スライド=1セクションを維持する。
- スライド画像リンクはそのまま残す。
- 日本語訳は直訳しすぎず、試験で理解しやすい表現にする。
- 理論ページでは、解説・試験で確実に理解すべき点・確認問題を追加する。
- 事務情報ページでは、解説を増やしすぎず、行動メモ中心にする。
- 数式はLaTeX記法で書く。
- `docs/03_past_exam_important_topics.md` を参照し、過去問に関連する箇所では年度・第何問・何を理解すべきかを明記する。
- 過去問は参考であり、試験対策ポイントには過去問以外の重要点も含める。

## 出力形式

各スライドを次の形式で出力してください。

```md
## Slide XXX: <title>

![Slide XXX](...)

### 日本語訳

### 解説

### 試験で確実に理解すべき点

### 過去問との関連

### 確認問題
```

## 注意

- 根拠のない断定をしない。
- 過去問との関連が薄いページでは「直接の対応は薄いが、講義全体の前提」と書く。
- 重要ページでは、答案で使える表現も入れてよい。
